SpotyReg

Описание

SpotyReg - это инструмент для автоматизации процесса регистрации аккаунтов spotify. Программа позволяет создавать множество аккаунтов в spotify с минимальным участием пользователя.

Основные возможности

      Автоматическая регистрация аккаунтов
      Экспорт данных в отдельный текстовый файл

Технические детали

Программа написана на Python с использованием современных библиотек для автоматизации и работы с сетью.
